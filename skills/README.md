# Skills

<p align="center">
  <strong>Text-only Agent Skills for Google AI Edge Gallery + Gemma 4 E2B</strong>
</p>

Each skill lives in its own `kebab-case` folder and consists of a single `SKILL.md` file.

### How to Import

In Google AI Edge Gallery:
1. Go to **Agent Skills**
2. Tap **+ Add Custom Skill**
3. Paste a raw URL:

```
https://raw.githubusercontent.com/Minocrisy/minocrisy-gemma-skills/main/skills/<skill-name>/SKILL.md
```
